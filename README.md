# CryptoTrack React App

This project was developed by using React Js(Javascript framework),Firebase and Coingecko API.

## Available features

### `Google SignIn`
### `Pagination`
### `User Authnecation`
### `Pagination and Searching`
### `Graph Representation as per Timestamps`
